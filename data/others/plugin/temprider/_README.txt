◇ 概要 ◇

このプラグインはティラノライダーの機能を拡張するプラグインです。
開発時にのみお使いいただき、リリース時にはパッケージからお外しください。



◇ ファイル構成 ◇

temprider
 ├ init.ks
 ├ main.js
 ├ view.html
 ├ _README1.png
 ├ _README2.png
 └ _README.txt



◇ 導入方法 ◇

temprider フォルダを
あなたのプロジェクトフォルダの data/others/plugin に配置して、
first.ksの適当なところに

    [plugin name="temprider"]

と記述することで使えます。
またこのとき、[plugin name="temprider" x="100" y="100"]のようにして、
いくつかの属性値を渡すことができます。

x          : テンプライダーウィンドウx座標　┐とちらも指定して
y          : テンプライダーウィンドウy座標　┘はじめて有効です
width      : テンプライダーウィンドウ横幅
height     : テンプライダーウィンドウ高さ
g_x        : ゲームウィンドウx座標　┐とちらも指定して
g_y        : ゲームウィンドウy座標　┘はじめて有効です
g_width    : ゲームウィンドウ横幅
g_height   : ゲームウィンドウ高さ
cache      : falseを指定すると、キャッシュを無効にします。
             →テキストエディタでシナリオを編集・保存してから
           　ゲームをロードすると即時に変更が反映されます。
temprider  : falseを指定すると、テンプライダーウィンドウを無効にします。

使用例

    [plugin name="temprider" x="120" y="0" g_x="460" g_y="0" g_width="960" g_height="640" cache="false" temprider="true"]

また、このプラグインを有効にするためには、
ティラノライダーウィンドウのその他タブの「デバッグを有効にする」に
チェックを入れてください。

◇

注意してほしいのですが、
このプラグインを導入した状態で
ゲームウィンドウの右上の更新ボタンを押すとバグります。
えらいことになるので、更新はゲームウィンドウ「左上」の更新ボタンか、
テンプライダーウィンドウの更新ボタンから行ってください。

2017/10/15  
・属性値入力欄において、
1)カーソルが数値の右側にある状態で上キーまたは下キーを押すと、
　数値が増減するように。
2)数値を選択した状態で上キーまたは下キーを押すと、
　数値が増減するように。
3)一部のタグ＋属性限定で、上キーまたは下キーを押すと、
