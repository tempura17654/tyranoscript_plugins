◇ 概要 ◇

ティラノスクリプトでメッセージ出力時にポポポ音を鳴らすようにするプラグインです。
音声ファイルを用意することなく、好きな音程でポポポポ慣らせます。
（もちろん音声ファイルを使うこともできます。）

なお、ポポポ音に興味はあるけどプラグインを入れるほどじゃねえべな、という場合、
こちらの記事の方法が参考になります。
【ティラノスクリプト】タイプライター風に文字を表示するマクロ - memo
http://7yen.hateblo.jp/entry/2016/12/02/021301



◇ ファイル構成 ◇

popopo
 ├ popopo.js
 ├ init.ks
 ├ _SAMPLE.ks
 └ _README.txt



◇ 導入方法 ◇

解凍して出てきた「popopo」フォルダを、
「data/others/plugin/」下にコピーしてください。
その後、first.ksに以下のタグを記述してください。

    [plugin name=popopo]

◇ タグリファレンス ◇

◆ [popopo] ポポポ音の設定変更
    
    || 概要
    
    ポポポ音の各種設定を変更します。
    音量、音声波のタイプ、使用する音声ファイル、音の高さ、時間、
    文字毎に鳴らすか一定間隔で鳴らすか、などなど。
    いろいろ変更できます。
    
    || 指定できるパラメータ
    
    type       : 7つのキーワードsine/square/sawtooth/triangle/noise/file/noneのいずれかで指定します。
                 順に、正弦波/矩形波/のこぎり波/三角波/ホワイトノイズ/音声ファイル再生/再生しない、を意味します。
    volume     : 音の大きさ。0～100の数値またはキーワードdefaultで指定します。
                 defaultを指定すると、コンフィグのSE効果音量を参照します。
    time       : ポポポ音の長さをミリ秒で指定します。type=fileの場合は無意味。デフォルトは20。
    tailttime  : ポポポ音のフェードアウト時間をミリ秒で指定します。type=fileの場合は無意味。デフォルトは30。
    
    frequency  : ポポポ音の音程。A/A+/B/B+/C/C+/D/D+/E/E+/F/F+/G/G+のいずれかのキーワードで指定します。
                 それぞれラ/ラ♯/……/ソ/ソ♯を意味します。なお、type=fileあるいはtype=noiseの場合は無意味。
    octave     : 音(オクターブ)の高さを整数で指定します。デフォルトは0。1増減させるごとに隣のオクターブに移動します。
                 type=fileあるいはtype=noiseの場合は無意味。
    
    samplerate : ポポポ音のサンプルレート。type=noiseの場合のみ機能します。初期値は44000。
    
    buf        : type=fileの場合のみ機能します。再生するスロットを整数で指定します。
    storage    : type=fileの場合のみ機能します。再生する音声ファイルを整数で指定します。
    
    mode       : everyone/intervalのどちらかのキーワードで指定します。
                 順に、ポポポ音を文字毎に鳴らす/文字に関係なく一定の間隔で鳴らす、を意味します。
                 デフォルトはeveryone。
    
    noplaychars: mode=everyoneの場合のみ機能します。ポポポ音を鳴らさない文字を指定できます。
                 デフォルトは"…・、。「」（）　 "。
    
    interval   : mode=intervalの場合のみ機能します。ポポポ音を鳴らす間隔をミリ秒で指定します。
                 デフォルトは80。
